## Brand Kit Setup (Do This First)

### Colors to Add in Canva Brand Kit:
| Name | Hex Code | Use |
|------|----------|-----|
| Coral (Primary) | #FF6B47 | CTAs, highlights, brand elements |
| Teal (Accent) | #14B8A6 | Secondary buttons, accents |
| Dark Text | #1C1917 | Headlines, body text |
| Light Gray | #FAFAFA | Backgrounds |
| Medium Gray | #78716C | Secondary text |
| White | #FFFFFF | Backgrounds, text on dark |

### Fonts to Add:
| Font | Use | Canva Alternative |
|------|-----|-------------------|
| Playfair Display | Headlines | Playfair Display (same) |
| Outfit | Body text | Outfit or Inter |
| Inter | UI elements | Inter (same) |

### Logo Variations Needed:
1. Full logo (horizontal)
2. Icon only (square)
3. White version (for dark backgrounds)
4. Black version (for light backgrounds)

## Design Tips for Maximum Impact

### For Pinterest:
- Use 2:3 ratio (1000x1500)
- Large, readable text
- Bright, eye-catching colors
- Include a clear CTA
- Add your website URL

### For Instagram:
- Consistent aesthetic
- Mix of product + lifestyle
- Use all feed formats (single, carousel, reels)
- Strong grid presence

### For Facebook:
- Lifestyle-focused imagery
- Emotional connection
- Social proof (testimonials, reviews)
- Clear value proposition

### For Ads:
- Clear, simple message
- One focal point
- Strong CTA
- Price/offer visible
- Brand colors consistent
